# AMSYNC
 
# Usage 
## Amplitude Modulation
- Process L and R Channel with amplitude Modulation 
- Interpolate between different Time Signatures 
- Synced to BPM
## Delay
- Process L and R Channel with delays
- Interpolate between different Time Signatures
- Synced to BPM
## Possible Effects
- Haas Effect
- Ping Pong Delay
- Panning
- ...
